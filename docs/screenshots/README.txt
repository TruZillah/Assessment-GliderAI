How to capture screenshots:
---------------------------
1. Open http://127.0.0.1:5000 in your browser
2. For dashboard.png: 
   - Show the main view with problem list and statistics
   - Use Windows Snipping Tool or Snip & Sketch (Win + Shift + S)

3. For editor.png:
   - Select a problem (like "Sum of Two Integers" or "Merge Intervals")
   - Show the code editor with some code
   - Expand the debugger or glossary
   - Capture the view

4. For tools.png:
   - Scroll to show the learning sections
   - Expand the Lecture section (click "Show lecture")
   - Expand the Cheat Sheet (click "Show cheat sheet")
   - Expand the Notepad (click "Show notepad")
   - Capture all the colorful learning tools visible together

5. Save all images as PNG format in this folder (docs/screenshots/)
